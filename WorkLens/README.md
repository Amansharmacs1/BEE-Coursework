# WorkLens

WorkLens is a minimalist, frontend-only Pomodoro-style timer and session tracker. Built entirely with vanilla HTML, CSS, and JavaScript, it offers a clean, modern, and intuitive user interface without relying on any external libraries or frameworks.

## Features

- **Customizable Intervals**: Set your preferred work and break durations.
- **Session Tracking**: Automatically cycles between Work and Break sessions.
- **Live Timer**: Displays remaining time, the current session type, and session number.
- **Detailed Statistics**: Tracks total work/break sessions, accumulated time for each, and the number of pauses taken.
- **Session History**: Maintains a log of completed and skipped sessions, showing start and end times.
- **Clean UI**: Beautiful, responsive, and minimalist design using vanilla CSS.

## How to Run

Since WorkLens is a purely frontend application with no build steps or dependencies, getting started is incredibly simple:

1. Clone this repository or download the source code.
2. Open `index.html` in any modern web browser.
3. Start focusing!

## Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Modern, minimal styling with custom variables, flexbox, and grid layouts.
- **Vanilla JavaScript**: Pure DOM manipulation and interval-based timer logic.
